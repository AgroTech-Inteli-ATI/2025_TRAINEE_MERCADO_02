# Fazenda Modelo Agrominas - Dashboard Interativa

## Sobre o projeto

Aplicação web para visualização do mapa da **Fazenda Modelo Agrominas**, com foco em:

- **SAF Café e SAF Abacate**
- **SSP (pecuária regenerativa)**
- **Compostagem**
- **Infraestrutura da fazenda**

O objetivo é oferecer uma interface simples e bonita para explorar áreas da fazenda, ver informações detalhadas e apoiar a gestão.

## Tecnologias

Este projeto é construído com:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Como rodar o projeto localmente

1. **Instale as dependências**:

```sh
npm install
```

2. **Execute o servidor de desenvolvimento**:

```sh
npm run dev
```

3. Abra o navegador em `http://localhost:8080` (ou a porta exibida no terminal).

## Build para produção

```sh
npm run build
```

## Estrutura principal

- `src/pages/FarmMap.tsx`: página do mapa interativo da fazenda.
- `src/components`: componentes reutilizáveis da interface.

Sinta-se à vontade para adaptar textos, cores e áreas do mapa conforme a realidade da sua fazenda.
